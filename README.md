# Spotify-Analytics with dbt

This project analyzes Spotify streaming and generates insights like top artists by popularity, genre, and audio features. It is an end-to-end data pipeline that pulls real music data from the Spotify Web API, processes and enriches it with Databricks on Azure, and loads it into Snowflake for analytics and visualization.

# Tech Stack
- Python ('spotipy', 'pandas')
- Azure
- Databricks
- Snowflake
